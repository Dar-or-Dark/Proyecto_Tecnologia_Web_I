Acceso remoto a Router Cisco

Configuracion Telnet


Primeramente se habilitara las lineas vty, que son canale de comunicación virtual para conectarse al dispositivo, creamos una contraseña y habilitamos el login


R1#(config)# line vty 0 15

R1#(config-line)# password cualquier_contraseña

R1#(config-line)# login

R1#(config-line)# end


Con eso ya podemos probrar la coneccion de remota desde cualquier dispositivo de la red, haciendo una coneccion a la ip del router con el comando telnet.


SW2# telnet 10.0.0.1


Configuracion SSH


Primeramente generamos una llave compartida en el router para poder realizar la coneccion


R1#(config)#crypto key generate rsa


Habilitamos el servicio y las lineas virtual vty p


R1#(config)#ip ssh version 2

R1#(config)#line vty 0 15

R1#(config)#transport input ssh

R1#(config)#login local


Creamos la credenciales de usuario


R1#(config)#username mario privilege 15 secret cisco123

R1#(config)#enable secret cisco123
