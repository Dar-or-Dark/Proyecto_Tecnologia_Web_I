Configuración de Telnet en Linux

Síguenos en Facebook, unite en Telegram y Suscribete en nuestro canal de Youtube

En la actualidad, Telnet no es un protocolo de comunicación remota demasiado utilizado para este tipo de soluciones, especialmente en entornos Linux, ya que han aparecido protocolos de comunicación mucho más seguros como SSH. Estos nos permiten establecer conexiones cifradas, mucho más seguras a ataques informáticos como spyware que Telnet.



Instalación y Configuración del Servidor Telnet


Instalar los paquetes del Servicio Telnet con los siguientes comandos:


sudo apt install openbsd-inetd


sudo apt install telnetd



Configurar y verificar del siguiente archivo


sudo nano /etc/inetd.conf


Dentro del archivo quitar la almohadilla o numeral (#), de la sección que describe:


#telnet stream tcp ……………….


Guardar los cambios efectuados Ctrl + x ó F3


Reiniciamos el Server Telnet instalado


sudo /etc/init.d/openbsd-inetd restart


Verificar si el Server Telnet logra escuchar.

(Verifica si está activo y en escucha)


netstat -ltp


(Verifica si está activo y en escucha además de mostrar porque puerto escucha el server telnet)


netstat ltpn
